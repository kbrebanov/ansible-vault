vault
=====

[![Build Status](https://travis-ci.org/kbrebanov/ansible-vault.svg?branch=ma
ster)](https://travis-ci.org/kbrebanov/ansible-vault)

Installs and configures Vault

Requirements
------------

This role requires Ansible 1.9 or higher.
